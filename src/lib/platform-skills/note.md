# Note — writing strategy

## Shape

Note mode is the freeform writing surface. No platform shape, no word target, no specific arc to land. Use it for journal entries, drafts that haven't found a platform yet, or quick captures that might become something later.

## What Thoughtbed should do in Note mode

The companion stays present but quieter:

- Surface Garden ideas the same way it does for other platforms.
- Don't push toward a hook / closer / word target.
- Don't suggest "publishing" the note — it's not a publishable shape.
- If the user starts shaping the note toward a specific platform (length picks up, structure emerges, hooks form), the companion can ask: "This is starting to read like a LinkedIn post. Want to switch the frame?"

## Voice

The same anti-AI rules apply. No em-dash pivots, no antithesis flips, no banned words. The note can be loose, but it shouldn't read like a machine.
